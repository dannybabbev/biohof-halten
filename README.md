# Biohof Halten

Marketing site for the Biohof Halten farm, built with Vite, React, TypeScript, Tailwind, and shadcn-ui.

## Prerequisites
- Node.js 18+ and npm

## Run locally
```sh
npm install
npm run dev
```

The dev server prints a local URL; open it in your browser to view the app.

## Production build
```sh
npm install
npm run build
```

The optimized output is emitted to `dist/`; serve the files from that directory with any static file server or your hosting of choice.
